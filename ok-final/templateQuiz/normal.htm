Quiz

Το παρακάτω quiz έχει στην ουσία τρία είδη quiz

 

Α) Ναι-Όχι, εδώ βάζουμε αντίστοιχα [Y] η [N] το Y η το N στα αγγλικά.

 

Β) Αντιστοίχισης, εδώ βάζουμε [A] [B] [C] [D] [E] [F] [G] η [0] εάν δεν αντιστοιχεί σε κάτι.

 

C) Συμπλήρωσης κενού, εδώ βάζουμε   [[λέξη]].

 

a.      Για να φτιάξουμε τα quiz ανοίγουμε το normal.html με το Word, δημιουργούμε τα quiz και σώζουμε σαν ιστοσελίδα  (htm,html)

 

b.      Το αρχείο normal.html εμπεριέχει σύνδεση στo JavaScript library jquery123.js και στο πρόγραμμα JavaScript  editey.js

 

c.       Επίσης όταν φτιάχνετε τα quiz να κάνετε παράβλεψη στα ορθογραφικά λάθη που εμφανίζει το Word, αλλιώς τα quiz ίσως να μην εμφανίζονται σωστά

 

d.      Για να δείτε τα quiz φορτώνεται την σελίδα normal.html στον browser (googlecrome-firefox)
